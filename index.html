FEATURE

アプリの解説

サイト名：
got gigs?
https://got-gigs.herokuapp.com

概要：
ミュージシャンとライブのマッチングサイト　※gigs=ライブ

使用言語など：
Ruby on Rails + Haml + SCSS(BEM) + jQuery + MySQL + AWS(S3)

説明：
・ライブ情報を投稿して、サポートミュージシャンを募集することができます。
・参加可能なミュージシャンとして、募集にエントリーすることができます。
・エントリー後は、メッセージを通してライブ参加への調整ができます。

制作背景：
バンドメンバー募集のサービスは多く存在するが、ライブ等で急遽必要になったサポートミュージシャンを募集するサービスがなかったため。
