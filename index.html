FEATURE

アプリの解説

サイト名：
got gigs?
https://got-gigs.herokuapp.com

概要：
ミュージシャンとライブ（gig）のマッチングサイト

使用言語など：
Ruby on Rails + Haml + SCSS(BEM) + jQuery + MySQL + AWS(S3)

説明：
・ライブ情報を投稿して、必要なミュージシャンを募集することができます。
・参加可能なミュージシャンとして、ライブにエントリーすることができます。
・エントリー後は、メッセージを通してライブに参加できるか調整できます。

制作背景：
バンドメンバー募集のサービスは多く存在するが、ライブで必要なミュージシャンを募集するサービスがなかったため。
